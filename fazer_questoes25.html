Computação Gráfica: AutoCAD, SketchUp e Revit

Computação gráfica é a área da informática responsável pela criação, manipulação e representação de imagens digitais, usadas em projetos arquitetônicos, engenharia e design. Programas como AutoCAD, SketchUp e Revit são ferramentas muito utilizadas para modelagem 2D e 3D, criação de plantas, maquetes digitais e projetos de construção.

AutoCAD: software para desenho técnico, muito usado para criar plantas e detalhes precisos em 2D e 3D.
SketchUp: ferramenta focada em modelagem 3D rápida e intuitiva, usada para criar maquetes e modelos conceituais.
Revit: software BIM (Building Information Modeling) que integra modelagem 3D com informações detalhadas para construção, facilitando o gerenciamento do projeto.

1. No AutoCAD, qual comando permite desenhar uma linha a partir de um ponto inicial até um ponto final definido pelo usuário?

2. No SketchUp, qual ferramenta é usada para extrudar superfícies e criar formas 3D a partir de desenhos 2D?

3. O Revit é considerado um software BIM. O que significa BIM?

Domínio de Windows, Word e Excel

O domínio de softwares básicos do pacote Microsoft Office e do sistema operacional Windows é fundamental em concursos públicos, pois são ferramentas muito utilizadas em rotinas administrativas.

Windows: sistema operacional que gerencia hardware e software, permitindo executar programas, organizar arquivos e configurar o ambiente de trabalho.
Microsoft Word: editor de texto usado para criar documentos formatados, como relatórios, cartas e trabalhos acadêmicos.
Microsoft Excel: programa de planilhas eletrônicas usado para organizar, calcular e analisar dados com fórmulas, gráficos e tabelas dinâmicas.

4. No Windows, qual atalho de teclado permite copiar um arquivo ou texto selecionado?

5. No Microsoft Word, qual função permite revisar as alterações feitas em um documento?

6. No Excel, qual fórmula deve ser usada para somar os valores das células A1 até A5?
